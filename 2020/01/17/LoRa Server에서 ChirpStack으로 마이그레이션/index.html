LoRa Server에서 ChirpStack으로 마이그레이션

Semtech사의 LoRa™️ 상표권 관련 문제로 LoRa Server 프로젝트가 ChirpStack으로 이름이 변경되었음.
https://forum.chirpstack.io/t/rename-to-chirpstack-announcement/6473

이에 따라 진행하던 프로젝트에 대한 수정사항

  • Repository 등록

  • Gateway Bridge 설치
    lora-gateway-bridge 에서 chirpstack-gateway-bridge 로 패키지 이름 변경됨

  • Network Server 설치
    loraserver 에서 chirpstack-network-server 로 패키지 이름 변경됨

  • Application Server 설치
    lora-app-server 에서 chirpstack-application-server 로 패키지 이름 변경됨

기타 파일 이름에 대한 변경사항이 모두 수정되었음
